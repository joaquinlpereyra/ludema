What is Ludema?

Ludema is a library intended to help beginner and intermediate programmers create terminal games. The project was started specifically with beginners in mind: if you have finished the Codecademy Python course, you know enough about Python and programming to use this library.

If you want to see how a game made with Ludema looks, here is a simple level of the classic Sokoban. The source code for this program is less than a 100 lines of code long after stripping comments!

Why should I use Ludema?

Ludema rides you of the most common complications when programming a game and leaves you to code only the fun parts. You get to decide the storyline, characters, challenges, enemies, maps and levels! Ludema only helps you by providing:

  • A dead simple way of creating maps.
  • Predefined but easily customizable classes for your Player, NPCs, Items and other Pieces.
  • Easy interfaces to make pieces move and attack. Creating your own is easy too!
  • Turns are automagically implemented to make enemies and other entities respond to the player.
  • A provided Screen class that is in charge of printing your game to the screen.
  • Clear exceptions that make it obvious where the problem was.
  • Emphasis on documentation: a method isn't a method until it has a clear and complete docstring.

import ludema

# We will our playable character
guy = ludema.pieces.Player(Fore.GREEN + "\u03A8", "Guy")

# and also a function that returns walls
def new_wall(): 
    return ludema.pieces.Wall(letter = ".")

# this is how we create a board
# first, lets define our blueprint
blueprint = """
. . . . . . . . 
. . * * * * * .
. . * . . @ * .
* * * . . . * .
* . . . . . * . 
* * * * * * *
""" 

# now we define a legend for our map blueprint
legend = {'*': new_wall,
          '@': guy}

# we define when the board is won and when is lost
# we will win whenever we move our guy once.
win_conditions = [lambda: len(guy.movements.history) == 1]
# and we will never lose
lose_contidition = [lambda: False]

# and now we can create our map
# everything that is not on the legend will be
# interpreted as empty space on the board
board = Board.new_from_blueprint("First", blueprint, legend,
                                 win_conditions, lose_conditions)

# only one step missing
# lets define a simple function to control our guy 
# with WASD keys, like FPS games do
def control_guy():
    mappings = {"w": guy.move.up,
                "s": guy.move.down,
                "d": guy.move.right,
                "a": guy.move.left,
                "q": sys.exit}

    action = user_input()
    try:
        mappings[action]()
    except KeyError:
        print("That's not an action. Try again")

# that's it, we can start our game:
screen = Screen(lambda: print(board), control_guy)
while True:
    screen.show(clear_after=True)
    if board.lost or board.won:
        break
