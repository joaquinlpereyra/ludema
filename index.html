What is Ludema?

Ludema is a library intended to help beginner and intermediate programmers create terminal games. The project was started specifically with beginners in mind: if you have finished the Codecademy Python course, you know enough about Python and programming to use this library.

If you want to see how a game made with Ludema looks, here is a simple level of the classic Sokoban. The source code for this program is less than a 100 lines of code long after stripping comments!

Why should I use Ludema?

Ludema rides you of the most common complications when programming a game and leaves you to code only the fun parts. You get to decide the storyline, characters, challenges, enemies, maps and levels! Ludema only helps you by providing:

  • A dead simple way of creating maps.
  • Predefined but easily customizable classes for your Player, NPCs, Items and other Pieces.
  • Easy interfaces to make pieces move and attack. Creating your own is easy too!
  • Turns are automagically implemented to make enemies and other entities respond to the player.
  • Clear exceptions that make it obvious where the problem was.
  • Emphasis on documentation: a method isn't a method until it has a clear and complete docstring.
